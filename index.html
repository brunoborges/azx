Java Home

Learn Java, find tools, discover libraries, run applications. Want to contribute? Hit the GitHub repo. PRs are welcome.

Install Java on Windows with Chocolatey: 

C:\> choco install openjdk

Install Java on Mac OS with Homebrew: 

$ brew install cask 
$ brew cask install java

Install Java on Ubuntu Linux: 

$ sudo apt-get install default-jdk

Write a HelloWorld.java 

public class HelloWorld {
    public static void main(String args[]) {
        System.out.println("Hello World!");
    }
}

Run it: 

java HelloWorld.java

What's next? Install an editor and learn the language.
